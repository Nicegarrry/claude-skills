The Proofing Room

A demo page for the drop-in review wrapper. Open it with ?proof to start.

What this is

This is ordinary page copy. With proofing mode on, you can pin a comment to any element, or edit the text in place, then export everything as JSON for an agent to action. Without the flag, this page looks completely normal.

Try editing this sentence in place — switch on “✎ Edit text”, click here, and rewrite it.

Things you can review

Headlines

Comment on tone, clarity, or length.

Body copy

Suggest rewrites inline without leaving the page.

Calls to action

Flag anything that feels off-brand.

Layout

Note spacing, hierarchy, or ordering issues.

Primary call to action

Far down the page

Scroll content exists below the panel so you can test that comment pins reposition on scroll, and that collapsing or dragging the panel lets you reach elements it was covering.

Add a comment here, collapse the panel to its toolbar, then drag it aside.
